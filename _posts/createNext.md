# Next.js とは

まず最初に Next.js とは何かについて。

Next.js は、React をベースに作られたオープンソースの Web アプリケーションフレームワークである。静的 Web サイト生成やサーバーサイドレンダリング（表示する Web ページの内容をサーバー側で生成してユーザーに送る手法）などの機能がある。

Next.js は、以下のサイトで公開されている。

https://nextjs.org/
